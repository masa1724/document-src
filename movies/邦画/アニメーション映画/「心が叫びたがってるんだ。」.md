# 心が叫びたがってるんだ。

### 概要
```
幼い頃、よくしゃべる女の子だったが、その何気なく発した言葉によって家族がバラバラに
なってしまった少女・成瀬順。
そこへ現れた卵の妖精に、二度と人を傷つけないようお喋りを封印され、喋るとおなかが痛くなる
呪いをかけられてしまう。それ以来、心を閉ざし、唯一のコミュニケーション手段は携帯メールのみとなってしまった。
高校2年生になった順はある日、担任から「地域ふれあい交流会」の実行委員に任命されるが....
```

### 感じた事
**・言葉はだれかを傷つける**
```
順が幼少時代のトラウマから発したセリフ。
確かに、言葉を発する(自分の意見を主張する)ことで相手と揉めたり、相手を傷つけることになるかもしれない。
だけど、拓実のいう通り言葉にしなければ相手が何を考えてるかわかるわけがないし、そんなやつに近づきたいと思う
人はほとんどいないと思う。

だからといって、裏でこそこそと人の文句や悪口を発することは当然よくない。
それは相手を傷つけることになるし、それを聞いている周りにいる人も不快にする。
そして、最後には自分自身を一番傷つけることになると思う(後悔しても取り戻せないし、その取り戻そうとする勇気を出すのも大変)。

一番やってしまいがちだと思ったのが、友人が他の人の悪口を言っている際にそれに同調してしまうことだと思う。
(いいだしっぺではなく、本当に思ってなかろうが、悪口を言っている人と同類。)
(仮に思っていても、相手がどんな気持ちで言っているのかを考えて、常に悪口・暴言にならないよう気を付けて発言していきたいと思う)

自分としては、相手に文句や不満があれば直接伝え、他人の悪口を言っている人には同調しない
ようにしていきたいと思う。
```

**・その人それぞれが何が出来て、どんなことが好きか・やりたいか見極める**
```
担任城嶋が最初に出した、ミュージカル案に大反対だった2年2組を見て思ったことで
どうすれば皆にミュージカル案を納得してもらえるか考えてみた。

まず、一口にミュージカルといってもやることは演技することだけでなく準備や衣装などやることは色々ある。
(演技、ストーリー作成、衣装作成、振付、ポスター看板、ステージ装飾、音声等、経理(どれでもあるが))

なので、こういう場合は、上記のような様々な仕事があるということを強調し、演技だけではないという
ことを伝え、それぞれにやりたい仕事を選んでもらう。
(個人的に、ミュージカルを嫌という人は、めんどくさいか人前での演技が嫌という人が多いと思うから。)

そこからそれぞれの仕事ごとにチームを作ることで皆で一つの物を作っているぞという気持ちを
感じてもらう。
その結果、配役の際もある程度自主的に参加したいと思う人が出てくるんじゃないかと
思った。(ここは微妙かもしれないが。。)

```
